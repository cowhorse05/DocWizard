# DocWizard Demo 示例

三个典型的大学生作业场景，帮助快速上手。

## 场景一：课程论文

**文件**：paper.md（含 Mermaid 架构图）+ references.bib

**任务**：Markdown → DOCX + PDF，Mermaid 图表渲染，参考文献格式化 (GB/T 7714)

**一句话**：「执行 task.md」

## 场景二：实验报告

**文件**：实验报告.md（含 LaTeX 公式）+ 实验数据.csv

**任务**：报告转换 + 数据分析 + 生成 PPT 汇报

**一句话**：「执行 task.md」

## 场景三：压缩包作业

**文件**：作业附件.zip → sales_data.csv + 作业要求.md

**任务**：解压 → 读取要求 → 数据分析 → 输出代码 + 报告(docx+pdf)

**一句话**：「执行 task.md」

---

每个场景目录下都有独立的 `task.md`，进入对应目录后对 AI 说「执行 task.md」即可。
